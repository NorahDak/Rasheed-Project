# Rasheed - Smart Exam Scheduling Assistant
Rasheed is an AI-powered academic assistant designed to help faculty members optimize exam schedules and minimize conflicts. By analyzing academic data, the system provides intelligent recommendations for selecting suitable exam dates, improving scheduling efficiency and enhancing the overall academic experience.

# Problem
Exam scheduling conflicts are among the major challenges faced by universities each semester. Manual scheduling often leads to overlapping exams and additional workload for faculty members.

# Solution
Rasheed assists faculty members by providing smart recommendations for exam dates based on academic data analysis, helping reduce scheduling conflicts and support better decision-making.

# Key Features
- Smart exam scheduling recommendations.
- Conflict reduction and optimization.
- User-friendly and intuitive interface.
- Seamless and efficient scheduling experience.
- AI-assisted decision support.

# Tools Used
- Figma
- Human-Centered Design (HCD)
- UX/UI Design Principles
